# Конфигурационный парсер — Вариант №24

Учебный инструмент командной строки, реализующий парсер и транслятор для **учебного конфигурационного языка**, описанного в варианте №24 задания по дисциплине *Конфигурационное управление* (РТУ МИРЭА, группа ИКБО-62-24).

Инструмент преобразует файл конфигурации, написанный на учебном языке, в **XML-представление** и выводит его в стандартный поток вывода.

---

##  Особенности языка (вариант 24)

- Поддержка **однострочных комментариев**: `-- это комментарий`
- Поддержка **многострочных комментариев**: `|# ... #|`
- Объявление констант: `def ИМЯ = значение`
- Использование констант: `?[ИМЯ]`
- Типы значений:
  - Целые числа (`[+-]?\d+`)
  - Строки в двойных кавычках (`"текст"`)
  - Массивы: `{ значение, значение, ... }`
- **Словари в этом варианте не поддерживаются** — вся структура строится через массивы и именованные константы.

---

##  Технологии

- Язык: **Python 3**
- Парсер: **[Lark](https://lark-parser.readthedocs.io/)** — современная библиотека для синтаксического разбора
- Генерация XML: встроенные модули `xml.etree.ElementTree` и `xml.dom.minidom`
